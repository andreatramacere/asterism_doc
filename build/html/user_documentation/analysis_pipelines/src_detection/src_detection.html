Introduction¶

This modules describe the SrcDetectionPipeline class used to create the pipeline for the source detection. The process in the pipeline are orchestrated by the do_src_detection_pipeline_func().

This pipeline has only one process:
  • DoSrcDetectionProcess (user guide: src_detection process user guide)

schematic view of the source detection pipeline
