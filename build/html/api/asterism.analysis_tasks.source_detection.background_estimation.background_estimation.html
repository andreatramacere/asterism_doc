Overview¶

This modules provides the implementation of the Task class DoSetBkgThreshTask used to estimate the background level in images. The Task algorithm is implemented by the func set_thresh()

Function to estimate the backgorund level in a given image.

  • If sub_block_frac_size is provided, the image is partitioned in N-blocks with size=sub_block_frac_size*image>size,

and the block with the lowest flux is used to estimate the bkg level. Otherwise, the full image is used. The first

-If skewness_range is provided the skewness is minimized using the function :function:`.minimize_skewness`, i.e. the minimum skewness in the range |m-s1,m+s1| is chosen, where m is the mean of the data, and s1=std(data)*`sigma_range`
