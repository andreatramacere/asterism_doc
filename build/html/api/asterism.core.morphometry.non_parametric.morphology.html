eval_Gini(flux_array)[source]¶

The Gini factor [Lotz:2004] indicates the distribution of the light among pixels. A value of Gini equal to 1 would indicate that all the light is concentrated in one pixel, on the contrary, a lower value would indicates that the light is distributed more evenly amongst the pixels, with Gini factor equal to 0, if all the pixels have the s ame flux value.

flux_array : 1dim numpy array,
