Module API¶

class AnalysisProcess(name, func, plot_func=None, parser=None, add_plot_task=True)[source]¶

Bases: object

This class is a base class to handle Processes. Processes are combinations of AnalysisTask. The combination of Tasks is implemented by the member func, a function that is implemented in each derived class, specific for each derived class, and that is passed as an argument of the constructor. The func is called by the method AnalysisProcess.run().
