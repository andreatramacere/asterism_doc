Module: analysis_pipeline¶

Overview¶

This module provides the implementation of the base class AnalysisPipeline class used to create specific Pipelines that combines together different AnalysisProcess. In the diagram the Pipeline #1 is combining different AnalysisProcess P1...Pn. See AnalysisPipeline for more details .
